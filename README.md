# 地图Key泄露测试工具

一个简单易用的地图API Key泄露检测工具，支持多种地图服务的有效性测试，提供直观的图形化界面和完整的响应内容展示。

## 📋 功能特性
- ✅ **多地图服务支持**：同时检测多个主流地图服务的API Key有效性
- 🎨 **美观的图形化界面**：基于 PyQt5 开发的现代化 GUI
- 📊 **详细的测试结果**：显示每个API的完整响应内容，便于分析
- ⚡ **异步测试**：测试在后台线程执行，界面保持响应
- 📝 **结果列表**：清晰显示每个服务的测试状态（有效/无效）
- 📁 **完整打包支持**：可打包为单个独立的EXE文件

## 🗺️ 支持的地图服务
- 高德地图
- 百度地图
- 腾讯地图

## 🚀 使用方法

### 使用图形化界面（推荐）
1. 双击start.bat一键启动
2. 运行 `main.py`
   ```bash
   python main.py
   ```
3. 在界面中输入API Key
4. 点击「开始测试」
5. 查看测试结果列表
6. 点击任意列表项查看完整响应内容

```bash

```

- 使用该工具请遵守相关法律法规和服务提供者的使用条款
